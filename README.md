# Afrobeats 2025 Archive

A curated visual archive of Afrobeats albums released in 2025.

## Features
- **Immersive Hero Section**: Auto-scrolling album marquee with glassmorphism effects.
- **Masonry Grid**: Responsive, "curated wall" layout with irregular card heights.
- **Genre Filter**: Filter albums by Afrobeats, R&B, Hip-Hop, etc.
- **Top 5 Curation**: Select your top 5 albums and export the list.

## Tech Stack
- React + Vite
- TypeScript
- CSS Modules / Vanilla CSS
- GSAP (Animation)

## Deployment

### Recommended: Vercel
1. pushed code to GitHub.
2. Go to [Vercel](https://vercel.com).
3. Click **Add New Project**.
4. Import the `Afrobeats25` repository.
5. Framework Preset should auto-detect as **Vite**.
6. Click **Deploy**.

Your live link will be ready in seconds!
